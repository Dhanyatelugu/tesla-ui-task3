# Tesla UI - Task 3 Mobile Prototype

## 📱 Project Overview
Tesla UI Task 3 is an interactive mobile prototype designed in Figma for the Thiranex UI/UX Internship. The project demonstrates a complete user journey for Tesla's mobile app with seamless navigation, smart animations, and a loop back to the homepage through a clickable prototype.

## ✨ Key Features
- **Interactive Navigation**: Smooth flow between 4 mobile screens
- **Smart Animate**: 300ms transitions with Ease Out easing for natural feel
- **Mobile-First Design**: Optimized for iPhone 13 Pro frame (390×844)
- **Clickable Prototype**: Functional "Order Now" & "Back to Home" buttons
- **User Journey Loop**: Complete flow from Homepage → Details → Thank You → Homepage
- **Professional Animations**: Consistent motion design across all screens

## 🔄 User Flow / Screens Included
1. **Homepage-Mobile** - Hero section with Tesla Model S preview
2. **Model S-Mobile** - Detailed specs and "Order Now" CTA
3. **Screen-2** - Order confirmation/details page  
4. **Screen-3** - Thank You page with "Back to Home" navigation
